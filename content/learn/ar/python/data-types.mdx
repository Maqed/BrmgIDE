## يعني ايه انواع البيانات اصلا؟

البيانات لازم تبقا ليها انواع عشان اللغة تعرف تفرق بينها. يعني مثلا ممكن يبقا فيه ارقام. فلازم اللغة تعرف تميز
عشان تعرف تتعامل مع الارقام دي. ولازم يبقا في نوع من البيانات عشان اللغة تعرف تتعامل مع الكلام المكتوب.
زي مثلا في المقدمة لما عملت متغير `name` وخليته `"Maged"`. هنشوف دلوقتي سوا الانواع دي.

## Int

Int هو العدد الفردي. زي في الماث الارقام -2,-1,0,1,2 وهكذا.
بايثون بقا بتتعامل مع الارقام الفردية لما تكتبها بالشكل دة

```python
age = 16 # لاحظ انا كتبت الرقم بشكل عادي خالص
```

# العمليات الرياضية

ممكن كمان نعمل عمليات رياضية زي الجمع والطرح.

```python
balance = 350
cost = 100
balance_after_purchase = balance - cost
print(balance_after_purchase) # جرب تطبعها بنفسك وشوف ايه اللي هيحصل
balance_after_refund = balance_after_purchase + cost
print(balance_after_refund)
```

```python
number_of_item = 3
cost = 100
total_cost = number_of_item * cost # بنستخدم النجمة عشان نضرب الارقام في بعض
print(total_cost)

cost_of_one_item = total_cost / number_of_item # بنستخدم السلاش (slash /) عشان نقسم
print(cost_of_one_item)
```

## Float

الFloat هو بيستخدم للارقام الكسر. زي مثلا 1.5, 3.14, 2.71. الخ..

```python
pi = 3.14
print(pi)
```

لو عايز اللغة تتعامل مع الرقم على انه رقم كسر حتى لو هو مش كسر في البداية لازم تكتبه فيه نقطة بعده

```python
cost = 100.0
print(cost)
```

طبعا تقدر تعمل عليها العمليات الرياضية زيها زي الـInt عادي خالص

# String

الـString هو نص. ممكن يكون اسم, عنوان. او اي حاجة تبقا كلام.

```python
name = "Maged"
print(name)
```

## العمليات على String

عادي تقدر تجمع وتضرب الـstrings في بعض. لكن متقدرش تطرحهم او تقسمهم

```python
address_one = "Fake Street"
address_two = "321"
full_address = address_one + " " + address_two # اضفت المسافة عشان ميبقوش لازقين في بعض
print(full_address) # جرب تطبع من غير المسافة
```

```python
symbol = "#"
print(symbol * 10)
```

جرب تحط الارقام في علمات تنصيص وتجمعهم.

```python
print("1" + "2")
```

هتطبع `12` عشان هو نص. مش هيطبع `3` زي في الارقام

# Boolean

النوع ده ليه قيمتين بس: `True` (صح) و `False` (غلط). وغالبا بنستخدمه في الشروط.
لاحظ ان اول حرف لازم يكون Capital.

```python
is_logged_in = True
has_permission = False
```
