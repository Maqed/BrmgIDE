## ايه هي بايثون؟

بايثون هي لغة متعددة الاستخدامات. ممكن تستخدمها في تطوير مواقع ويب, برامج ديسكتوب.
ذكاء اصطناعي, الخ...

## طيب, ايه هي الاساسيات؟

في المقدمة دي هنتعلم ابسط الحاجات. ازاي تعمل variable (اللي هو متغير) وتطبعه.

```python
name = "Maged"
print(name)
```

جرب تغير اللي جوا الـ`name` وتخليه اسمك وتطبعه بنفسك في معدل الاكواد (IDE او Code editor).
