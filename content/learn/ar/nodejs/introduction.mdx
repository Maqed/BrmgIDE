# مقدمة إلى Node.js

## ما هو Node.js؟

Node.js هو بيئة تشغيل JavaScript قوية ومفتوحة المصدر تتيح لك تشغيل كود JavaScript خارج متصفح الويب. بُني Node.js على محرك V8 JavaScript من Chrome، ويمكّن المطورين من استخدام JavaScript لبرمجة الخادم، مما يجعل من الممكن بناء تطبيقات متكاملة باستخدام لغة برمجة واحدة فقط.

## لماذا نستخدم Node.js؟

### 1. **JavaScript في كل مكان**

مع Node.js، يمكنك استخدام JavaScript لكل من تطوير الواجهة الأمامية والخلفية، مما يقلل من تعقيد تعلم لغات متعددة ويسمح بإعادة استخدام الكود بشكل أفضل.

### 2. **إدخال/إخراج غير محجوب**

يستخدم Node.js نموذجاً مدفوعاً بالأحداث وإدخال/إخراج غير محجوب، مما يجعله خفيفاً وفعالاً. هذا مفيد بشكل خاص لبناء تطبيقات شبكية قابلة للتوسع تتعامل مع العديد من الاتصالات المتزامنة.

### 3. **تنفيذ سريع**

بفضل محرك V8، يقوم Node.js بتحويل JavaScript مباشرة إلى كود آلة أصلي، مما يؤدي إلى أوقات تنفيذ سريعة.

### 4. **نظام بيئي غني**

يوفر npm (Node Package Manager) الوصول إلى أكثر من مليون حزمة، مما يسهل إضافة وظائف إلى تطبيقاتك دون الحاجة لإعادة اختراع العجلة.

### 5. **مجتمع نشط**

يمتلك Node.js مجتمعاً كبيراً ونشطاً يساهم في نظامه البيئي، ويقدم الدعم، ويحسن المنصة باستمرار.

## الميزات الرئيسية

- **غير متزامن ومدفوع بالأحداث**: جميع واجهات برمجة التطبيقات في Node.js غير متزامنة، مما يعني أن الخادم لا ينتظر واجهة برمجة التطبيقات لإرجاع البيانات قبل الانتقال إلى المهمة التالية.
- **أحادي الخيط**: يستخدم Node.js نموذجاً أحادي الخيط مع حلقة أحداث، مما يجعله قابلاً للتوسع بشكل كبير.
- **متعدد المنصات**: اكتب مرة واحدة، شغّل في أي مكان - يمكن تشغيل تطبيقات Node.js على Windows وLinux وmacOS.
- **بدون تخزين مؤقت**: تخرج تطبيقات Node.js البيانات على شكل أجزاء، ولا تخزن أي بيانات مؤقتاً أبداً.

## حالات الاستخدام الشائعة

يتفوق Node.js في سيناريوهات مثل:

- **واجهات برمجة تطبيقات RESTful**: بناء خوادم API خفيفة وفعالة
- **التطبيقات في الوقت الفعلي**: تطبيقات الدردشة، الإشعارات المباشرة، أدوات التعاون
- **تطبيقات البث**: معالجة تدفقات البيانات مثل الفيديو أو الصوت
- **الخدمات الصغيرة**: إنشاء خدمات صغيرة ومستقلة تعمل معاً
- **أدوات سطر الأوامر**: بناء أدوات CLI وأدوات التطوير

## برنامجك الأول في Node.js

إليك مثال بسيط "Hello World":

```javascript
// أنشئ ملفاً باسم app.js
console.log("Hello, Node.js!");
```

قم بتشغيله باستخدام:

```bash
node app.js
```

## إنشاء خادم بسيط

يسهّل Node.js إنشاء خادم ويب:

```javascript
const http = require("http");

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader("Content-Type", "text/plain");
  res.end("Hello World from Node.js!");
});

const PORT = 3000;
server.listen(PORT, () => {
  console.log(`Server running at http://localhost:${PORT}/`);
});
```

احفظ هذا كـ `server.js` وشغّله باستخدام `node server.js`. ثم افتح متصفحك على `http://localhost:3000` لرؤية خادمك يعمل!

## الخطوات التالية

الآن بعد أن فهمت أساسيات Node.js، أنت مستعد لـ:

- تعلم npm وإدارة الحزم
- استكشاف نظام الوحدات في Node.js
- فهم البرمجة غير المتزامنة باستخدام callbacks و promises و async/await
- بناء تطبيقك الأول في العالم الحقيقي

يفتح Node.js عالماً من الإمكانيات لمطوري JavaScript. سواء كنت تبني واجهات برمجة التطبيقات، أو تطبيقات في الوقت الفعلي، أو أدوات سطر الأوامر، فإن Node.js يوفر الأدوات والأداء الذي تحتاجه للنجاح.
