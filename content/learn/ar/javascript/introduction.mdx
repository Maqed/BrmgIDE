## ايه هي جافاسكريبت؟

جافاسكريبت (JavaScript) هي لغة برمجة بتستخدم بشكل أساسي في تطوير الويب عشان تخلي المواقع تفاعلية. كمان ممكن تستخدمها في السيرفر (Node.js) وتطبيقات الموبايل وغيرها.

## طيب, ايه هي الاساسيات؟

في المقدمة دي هنتعلم ابسط الحاجات. ازاي تعمل variable (اللي هو متغير) وتطبعه في الـ console.

```javascript
let username = "Maged";
console.log(username);
```

جرب تغير اللي جوا الـ`username` وتخليه اسمك وتطبعه بنفسك في معدل الاكواد (IDE او Code editor) أو في الـ Console بتاع المتصفح.
