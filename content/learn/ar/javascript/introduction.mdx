# مقدمة إلى JavaScript

## ما هي JavaScript؟

JavaScript هي لغة برمجة عالية المستوى ومفسرة تجعل صفحات الويب تفاعلية. تسمح لك بتنفيذ ميزات معقدة على صفحات الويب، مثل تحديثات المحتوى الديناميكية، والخرائط التفاعلية، والرسومات المتحركة ثنائية وثلاثية الأبعاد، والمزيد. إنها الطبقة الثالثة من كعكة تقنيات الويب القياسية، حيث غطينا الطبقتين الأخريين (HTML و CSS) بتفصيل أكبر في أجزاء أخرى.

## لماذا تتعلم JavaScript؟

### 1. **لغة البرمجة الأكثر شعبية**

JavaScript هي لغة البرمجة الأكثر شعبية في العالم. إنها لغة الويب، وتستخدمها أكثر من 97% من جميع مواقع الويب.

### 2. **متعددة الاستخدامات**

يمكن استخدام JavaScript لتطوير الواجهة الأمامية (React, Vue, Angular)، وتطوير الخلفية (Node.js)، وتطوير تطبيقات الجوال (React Native)، وحتى تطوير تطبيقات سطح المكتب (Electron).

### 3. **صديقة للمبتدئين**

JavaScript لغة رائعة للمبتدئين لأنها سهلة البدء. يمكنك تشغيل كود JavaScript مباشرة في متصفحك دون إعداد أي بيئة تطوير.

### 4. **مجتمع رائع**

تمتلك JavaScript مجتمعاً ضخماً من المطورين المستعدين دائماً للمساعدة. هناك الكثير من الموارد المتاحة عبر الإنترنت لمساعدتك في تعلم JavaScript.

## الميزات الرئيسية

- **لغة مفسرة**: يتم تنفيذ كود JavaScript سطراً تلو الآخر بواسطة محرك JavaScript في المتصفح.
- **برمجة من جانب العميل**: تعمل JavaScript على جهاز العميل (في المتصفح)، مما يعني أنها يمكن أن تتفاعل مع إجراءات المستخدم فوراً دون التواصل مع الخادم.
- **مدفوعة بالأحداث**: يمكن لـ JavaScript الاستجابة لأحداث المستخدم مثل النقرات وحركات الماوس وضغطات المفاتيح.
- **كائنية التوجه**: تدعم JavaScript مبادئ البرمجة كائنية التوجه (OOP).

## برنامجك الأول في JavaScript

يمكنك تشغيل كود JavaScript مباشرة في وحدة تحكم المتصفح الخاص بك. افتح أدوات المطور في متصفحك (F12)، وانتقل إلى علامة التبويب Console، واكتب:

```javascript
console.log("Hello, World!");
```

أو يمكنك تضمينه في ملف HTML:

```html
<!DOCTYPE html>
<html>
  <body>
    <h1>برنامجي الأول في JavaScript</h1>
    <script>
      console.log("Hello, World!");
    </script>
  </body>
</html>
```

## الخطوات التالية

الآن بعد أن عرفت ما هي JavaScript، أنت مستعد لـ:

- تعلم المتغيرات وأنواع البيانات
- فهم المعاملات والتعبيرات
- إتقان التحكم في التدفق (if/else, loops)
- الغوص في الدوال والكائنات

JavaScript لغة قوية تفتح العديد من الفرص. برمجة سعيدة!
